WangStyle 是一款极简的、支持暗色模式的 `typecho` 后台美化插件。

### 插件特点

1. 默认简白，支持暗色模式
2. 不同级的用户登录显示不同的菜单
3. 自动添加插件设置到菜单
4. 即插即用、不修改程序文件
5. 响应式布局，适配后台每一个页面
6. 后台登陆界面美化
7. 后台首页卡片样式

### 注意事项

1. 本插件只针对 `Typecho` 默认的程序美化。
2. 启用插件后，CTRL + F5 清除缓存。
3. 插件名为：WangStyle，注意区分大小写，否则500。
